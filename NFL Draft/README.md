# NFL Draft

2025 NFL mock draft consensus tracker. Aggregates mock draft picks from 30+ analysts to identify consensus draft positions and compare player projections.

## Files

| File | Description |
|------|-------------|
| `NFL Draft Analysis.R` | Parses consensus tracker CSV, extracts player names/positions/colleges. Includes function to compare mock positions between two players across multiple draft boards. |
| `2025 NFL Mock Draft_Big Board Consensus Tracker - Mock Drafts 2025.csv` | Multi-header CSV with picks from 30+ sources (Sharp, ESPN, Walter Football, NFL.com, Brooks, etc.) |

## Dependencies

- `tidyverse`, `readr`
